# BackendL5

BackendL5 for Laravel 5 based in [AdminLTE template](http://almsaeedstudio.com/AdminLTE/).

###Install

Copy file .env.example to .env

```
cp .env.example .env
```

Edit .evn and change environment vars.

```
vim .env
```

Migrate tables

```
php artisan migrate
```

Install users

```
php artisan db:seed
```
