# BackendL5

BackendL5 for Laravel 5 based in [AdminLTE template](http://almsaeedstudio.com/AdminLTE/).

###Install

Copy file .env.example to .env and edit for change env vars.

###Migrate

Migrate db

```
php artisan migrate
```

Install users

```
php artisan db:seed
```
